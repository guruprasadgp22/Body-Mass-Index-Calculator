# 📊 Body Mass Index (BMI) Calculator

A simple and responsive **BMI Calculator** built using **HTML**, **CSS**, and **JavaScript**. This tool helps users calculate their Body Mass Index based on their gender, age, height, and weight, providing an instant BMI value along with the weight category.

---

## 🚀 Features

- 📏 Calculates BMI based on user inputs (age, gender, height, weight)
- 📊 Displays BMI result and weight category (Underweight, Normal, Overweight, Obese)
- 🎨 Clean and responsive UI design
- ⚡ Instant calculation without page reload

---

## 🛠️ Technologies Used

- **HTML** for structure
- **CSS** for styling
- **JavaScript** for dynamic functionality

---

## ⚙️ Installation Process

1. **Clone the repository:**  
   ```bash
   git clone https://github.com/your-username/bmi-calculator.git
   ```

2. **Navigate to the project directory:**  
   ```bash
   cd bmi-calculator
   ```

3. **Run the project:**  
   Open the `index.html` file in your preferred browser:
   - Double-click `index.html` **or**
   - Right-click `index.html` → Open with → Your Browser

---

## 📝 Usage

1. Select your **gender** from the dropdown.
2. Enter your **age**, **height** (in feet and inches), and **weight** (in kg).
3. Click the **"Calculate BMI"** button.
4. View your **BMI result** and **weight category** below the button.
